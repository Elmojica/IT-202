Apple.com

This page is appealing because...

  1. There is no clutter, as you scroll down the website, you'll see a section for each of the lastest products that apple is selling
  2. The website is intuitive, all of the necessary options are given on the top nav bar. This allows the user to navigate to the desired page faster
  3. The imagery used for each section correctly reflects apples brand as well as the product that it is trying to showcase.

I would definitely recommend others to use this website, either to purchase apple products or to use the website for inspiration to design their own websites. This website doesnt overwhelm the user with too many options while also allowing them access their desired page faster via the nav bar.

Craiglist

This page is unappealing because...

  1. It is far too cluttered, and if you click on the wrong link, you'll have to navigate back to the home page to get to where you need to be.
  2. It only has text, making use of some imagery or colors could greatly improve the overall look of this website.
  3. Using broader categories on a nav bar can allow users to navigate throughout the website alot more easily.

Suggestions for Improvement:

  1. Separate each different category and have a main page listing those main categories.
  2. Add more user friendly ui, as in, maybe a main nav bar that can expand when you hover on it and broader sections on the start page with images that reflect.
  3. Use brand specific colors to maintain the reputation of the brand and allow the page to be more recognizable.

I would not encourage others to visit this website because of the difficulty that comes with trying to maneuver around this website to find what you're looking for.

Arngren

This page is unappealing because...

  1. Too much information is being shown to you all at once
  2. It takes a while to load all the different images. In other words, there are too many images trying to catch the users attention
  3. The front page isnt organized at all, things seem to be listed wherever they fit on the page.

Suggestions for Improvement:

  1. Separate each of the listed items into broader categories and list them on a navigation bar.
  2. Add more whitespace when listing each individual purcheasable item to avoid any confusion from the users end.
  3. Use more uniform spacing when listing each items. Perhaps by using a grid-like format.

I would not recommend using this website because of how confusing the UI design is. It seems as if the listed items were placed with little to no thought into their placement. It seems like they wanted to fit as many things on the main page as possible to allow the user to see as many items as possible.
